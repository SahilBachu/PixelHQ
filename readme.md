# PixelHQ

A single-player AI agent management game built with Phaser 3 and React.

## Features

- Pixel-art virtual office environment
- AI agent management with status monitoring
- Glassmorphic dark UI with left/right sidebars
- Character selection and movement (WASD/arrow keys)
- Mobile joystick support

## Controls

- `W, A, S, D, or arrow keys` to move
- `E` to sit down / stand up
- `R` to interact with objects

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/en/) 16+
- npm or yarn

### Installation

```bash
cd client
npm install
npm run dev
```

The dev server will start at `http://localhost:5173`.
